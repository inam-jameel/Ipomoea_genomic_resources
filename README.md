# Ipomoea Genomic Resources

A curated collection of links and resources for Ipomoea genomic research and data.

## Overview

This GitHub Pages site serves as a centralized hub for accessing external genomic resources, databases, and tools related to Ipomoea species. Visit the site here: [Ipomoea Genomic Resources](https://inam-jameel.github.io/Ipomoea_genomic_resources/)

The site is currently under onstruction, so many links are broken and are in development

## Contents

The website will provide quick access to:
- Genomic databases
- Research tools and software
- Reference sequences
- Related publications
- Community resources

## Quick Links

Browse the full collection by visiting the [main page](https://inam-jameel.github.io/Ipomoea_genomic_resources/).

## About Ipomoea

Ipomoea is a diverse genus of flowering plants in the family Convolvulaceae, commonly known as morning glories. This collection focuses on genomic and molecular resources for research purposes.
